#  Modern Weather App

A sleek and modern Weather Application built using Python and CustomTkinter. This app fetches real-time weather data and displays it with a beautiful dark-themed UI.

---

# Features

- 🌍 Search weather by city name
- 🌡️ Current temperature (°C / °F toggle)
- ☁️ Weather condition & description
- 💧 Humidity, wind speed, pressure
- 📅 5-day weather forecast
- 🌐 Auto location detection
- 🎨 Modern dark UI with glassmorphism design
- ⚡ Fast and responsive interface

---

# Tech Stack

- Python
- CustomTkinter
- Requests (API calls)
- Pillow (Image handling)
- OpenWeatherMap API
